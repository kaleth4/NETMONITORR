## рҹҡЁ ClasificaciГіn de riesgo
 
El script evalГәa automГЎticamente cada conexiГіn y le asigna un nivel de riesgo:
 
| Nivel | Prefijo | DescripciГіn |
|-------|---------|-------------|
| вң… **INFO** | `   ` | Puertos estГЎndar sin riesgo conocido |
| рҹ”ө **BAJO** | `   ` | Puertos del sistema sin amenaza activa |
| рҹҹЎ **MEDIO** | `   ` | Servicios comunes que merecen atenciГіn |
| рҹҹ  **ALTO** | `!  ` | Servicios potencialmente explotables |
| рҹ”ҙ **CRГҚTICO** | `!! ` | Puertos asociados a ataques activos |
